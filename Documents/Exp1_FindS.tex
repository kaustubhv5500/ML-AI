    \section{What is Find-S Algorithm?}
    \textbf{Find-S} is a \textbf{basic concept based machine learning algorithm} which tries to find the \textbf{most specific hypothesis} that fits all \textbf{positive examples}.It starts with the most specific hypothesis and generalizes it each time it fails to classify an observed positive training data. The algorithm is used to find the \textbf{maximally specific hypothesis} and forms the starting point for various decision algorithms in machine learning.
    
\subsection{Objective}To explore how \textbf{FIND-S algorithm} is used for finding the \textbf{most specific hypothesis} that fits all positive examples based on a given set of training data samples. A \textbf{positive example} or \textbf{positive training instance} refers to a positive outcome when considering the binary target concept attribute of the dataset.

\subsection{Algorithm}
\begin{enumerate}
    \item Initialize \textit{h} to the \textbf{most specific hypothesis} in \textit{H}, where \textit{H} is the \textbf{initial dataset}.
    \item In each \textbf{positive training instance} \textit{x} 
    \begin{itemize}
    \item For each \textbf{attribute constraint} $a_i$ in \textit{h}.
    \item If the \textbf{constraint} $a_i$ is satisfied by \textit{x}.
    \item Then do nothing.
    \item Else replace $a_i$ by the next \textbf{more general constraint} that is satisfied by \textit{x}. 
    \end{itemize}
    \item Output \textbf{hypothesis} \textit{h}.
\end{enumerate}

\subsection{Procedure}
\begin{enumerate}
    \item Representation of \textbf{hypothesis}.
     \item Apply \textbf{Find-S algorithm} on the given data to get the \textbf{most specific hypothesis}.
     \item Interpret the output of \textbf{Find-S}.
\end{enumerate}

\subsection{Dataset Summary}
The dataset is a weather dataset containing attributes that determine whether an individual will play a sport on that day or not. The target concept is the \textbf{Sport} attribute which is a binary value.

\begin{itemize}
    \item Number of Instances: 8 
\item Number of Attributes: 9 (including the class attribute) 
 \item Attribute Information:
 \begin{enumerate}
    \item Time: Morning or Evening (Time for playing)
    \item Sky: Sunny or Rainy
    \item AirTemp: Warm or Cold
    \item Humidity: Normal or High
    \item Wind: Strong or Mild
    \item Weather: Warm or Cool
    \item Company: Yes or No
    \item Forecast:  Same or Change
    \item Sport: Yes or No
 \end{enumerate}
\end{itemize}
